## 1. Главный координационный чат
Задача:
- удерживать архитектурную целостность проекта
- координировать работу между чатами
- определять, затрагивает ли задача критичные слои
- не допускать конфликтов между зонами

## 2. Чат публичной части / storefront
Задача:
- главная страница
- каталог
- карточка товара
- хедер, футер, layout
- публичные статические страницы

Не трогает:
- admin-логику
- core-контракты без согласования
- общие типы без необходимости

## 3. Чат admin / управление товарами
Задача:
- список товаров
- создание и редактирование товара
- удаление товара
- скрыть / показать на сайте
- загрузка изображений
- admin UX

Не трогает:
- публичную часть сайта
- глобальные контракты без согласования
- общую архитектуру без необходимости

## 4. Чат core / architecture / data contracts
Задача:
- модель Product
- общие типы
- shared helper logic
- slug, parse, validate
- data flow
- критичные API-контракты

## 5. Чат SEO / content / commercial pages
Задача:
- тексты
- SEO-структура
- meta title / description
- коммерческие страницы
- FAQ и контентные блоки
