# PROJECT_CONTEXT

## Проект
Сайт по продаже и выкупу б/у техники Apple по всей России с основным фокусом на Москву.

## Основная идея
Нужен современный, удобный и понятный сайт, который сочетает:
- каталог б/у техники Apple
- карточки товаров с подробными характеристиками
- направление выкупа / trade-in
- доверительные и коммерческие страницы
- в будущем — возможность расширения до дополнительных сервисов

## Цель запуска
Собрать проект с нуля, но уже без старых архитектурных ошибок, с понятной структурой, разделением зон ответственности и возможностью вести разработку через несколько отдельных чатов.

## Базовый стек
- Next.js
- React
- TypeScript
- Tailwind CSS

## Приоритеты
1. Понятная архитектура
2. Удобство дальнейшей разработки
3. Современный дизайн
4. Хорошая структура каталога
5. Отдельный сценарий для выкупа / trade-in
6. Масштабируемость проекта

## Примечание
Этот проект не должен копировать сайты-конкуренты. Допустимо брать только общую бизнес-логику и полезные сценарии, но реализация должна быть лучше по структуре, дизайну и удобству.
